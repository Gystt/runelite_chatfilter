# runelite_chatfilter
Regex for the Chat Filter plugin of Runelite

INSTALLATION:
If you need to download Runelite, then you can do so here: https://runelite.net/<br>
On Runelite, click the spanner icon at the top right. Next go down to Chat Filter, and make sure it's turned on, then click on the cog wheel to see the options. Now copy and paste all of regex.md into Filtered Regex, and click back into your game to see the effects.
